# AI DevDocs Assistant - Client

The frontend dashboard for the AI DevDocs Assistant, built with React and Vite.

## 🚀 Features
- **File Upload**: Interface for PDF and Swagger files.
- **Chat Interface**: Real-time AI chat with documentation context.
- **Document Management**: View and track processing status of uploaded docs.

## 🛠️ Setup
1. Create `.env` and set `VITE_API_URL`.
2. Install dependencies: `npm install`.
3. Run dev server: `npm run dev`.

## 📂 Structure
- `src/App.tsx`: Main dashboard and chat logic.
- `src/api.ts`: API service for backend communication.
- `src/types.ts`: TypeScript definitions.
